MRZ Checksum Validation Algorithm

1
แปลงตัวอักษร A-Z → 10-35, 0-9 → 0-9, < → 0
2
คูณแต่ละตัวด้วย weight pattern: 7, 3, 1, 7, 3, 1...
3
รวมผลคูณทั้งหมด: A(10)×7 + A(10)×3 + 1×1 + 2×7 + ...
4
หาร mod 10: sum % 10 = check digit
✅ Passport No. Check: AA1234567 → check digit = 8 — VALID
✅ DOB Check: 900115 → check digit = 1 — VALID
✅ Expiry Check: 261231 → check digit = 1 — VALID
✅ Composite Check Digit: 04 — ALL VALID
