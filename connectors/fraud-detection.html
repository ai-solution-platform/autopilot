AI Detection Methods (8 เทคนิค)
🔍
Metadata Analysis
ตรวจสอบ EXIF, creation date, software used เพื่อหาความผิดปกติในข้อมูล metadata ของเอกสาร
🔬
Pixel-Level Analysis
ตรวจหาร่องรอยการแก้ไขรูปภาพ เช่น noise inconsistency, compression artifacts
🔤
Font Consistency
ตรวจสอบความสม่ำเสมอของฟอนต์ทั้งเอกสาร รวมถึง kerning และ weight
💧
Watermark Detection
ตรวจสอบลายน้ำ / ตราประทับ ว่ามีอยู่จริงและไม่ถูกแก้ไข
📊
Template Matching
เปรียบเทียบกับ template เอกสารจริง ของสถาบันการเงินต่างๆ
🔗
Cross-Reference
เทียบข้อมูลกับฐานข้อมูลอ้างอิง เช่น ชื่อบริษัท เลขบัญชี ข้อมูลสาขา
✏️
Signature Analysis
ตรวจสอบลายเซ็น ด้วย biometric comparison กับฐานข้อมูลลายเซ็น
🤖
AI Deep Learning
โมเดล AI ที่เรียนรู้จากเอกสารจริง/ปลอม กว่า 500,000 ฉบับ ด้วย Neural Network
