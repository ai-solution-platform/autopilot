01 — Pain Points

ปัญหาที่พบ

ปัญหาหลักที่องค์กรเผชิญในกระบวนการจัดซื้อและวิเคราะห์ใบเสนอราคา

📑

เอกสารหลากหลายรูปแบบ

PDF, Scan, ภาพถ่าย แต่ละผู้จำหน่ายส่งเอกสารคนละ Format ทำให้ยากต่อการรวบรวมและเปรียบเทียบ

⏰

ใช้เวลามากในการรวบรวมใบเสนอราคา

ต้องเปิดเอกสารทีละไฟล์ คัดลอกข้อมูลลง Excel ทำให้เสียเวลาหลายชั่วโมงต่อรอบจัดซื้อ

⚠️

โอกาสผิดพลาดสูงจากการกรอกข้อมูลด้วยมือ

ข้อมูลราคา จำนวน หน่วย ที่กรอกด้วยมือมีโอกาสผิดพลาด ส่งผลต่อการตัดสินใจ

🗄️

ไม่มีระบบจัดเก็บเอกสารที่เป็นระบบ

เอกสารกระจายอยู่ใน Email, แชท, โฟลเดอร์ส่วนตัว ค้นหาย้อนหลังได้ยาก

📊

การสร้างตารางเปรียบเทียบด้วยมือ

ต้องสร้างตารางเปรียบเทียบราคาใน Excel ด้วยตัวเอง ใช้เวลานานและไม่ได้มาตรฐาน

🔍

ขาดความโปร่งใสในกระบวนการจัดซื้อ

ไม่มี Audit Trail ที่ชัดเจน ยากต่อการตรวจสอบและอธิบายเหตุผลในการเลือกผู้จำหน่าย

05 — AI Technology

AI & LLM Technology

เทคโนโลยี AI ที่ขับเคลื่อน Autopilot Platform

👁️

OCR Engine

เครื่องมือแปลงเอกสาร Scan เป็นข้อมูลดิจิทัล

  • Tesseract OCR สำหรับเอกสาร Scan
  • Google Cloud Vision สำหรับ Complex layouts
  • รองรับ PDF, JPEG, PNG, TIFF
  • ความแม่นยำ 98.7% สำหรับเอกสารดิจิทัล
🧠

LLM Integration

Large Language Model สำหรับวิเคราะห์และสกัดข้อมูล

  • OpenAI GPT-4 สำหรับ Intelligent extraction
  • Claude สำหรับ Complex analysis
  • Structured output — JSON / Table format
  • Multi-step reasoning สำหรับเอกสารซับซ้อน
🇹🇭

Thai NLP

รองรับภาษาไทยเต็มรูปแบบ

  • Thai text segmentation & tokenization
  • เข้าใจรูปแบบใบเสนอราคาไทย
  • Extract ราคา, หน่วย, รายละเอียดสินค้า
  • รองรับทั้งเอกสารไทยและอังกฤษ
🏷️

Document Classification

จำแนกประเภทเอกสารอัตโนมัติ

  • Auto-detect: Quotation, Invoice, PO
  • Route เอกสารไปยัง Pipeline ที่ถูกต้อง
  • Confidence scoring สำหรับการจำแนก
  • Human-in-the-loop สำหรับ Low confidence
