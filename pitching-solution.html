Deployment Options

รูปแบบการให้บริการ

เลือกรูปแบบที่เหมาะสมกับนโยบายความปลอดภัยขององค์กร

On-Premise

ติดตั้งบน Server องค์กร

ข้อมูลทั้งหมดอยู่ภายในองค์กร 100% เหมาะกับองค์กรที่มีข้อกำหนดด้านความปลอดภัยสูง

จุดเด่น

  • ข้อมูลทั้งหมดอยู่ภายในองค์กร
  • เหมาะสำหรับองค์กรที่มีข้อกำหนดด้านความปลอดภัยข้อมูลสูง
  • ปรับแต่งระบบให้สอดคล้องกับระบบภายในองค์กรได้

ข้อพิจารณา

  • ต้องมีโครงสร้างพื้นฐานด้าน Server
  • ต้องมีการดูแลระบบโดยทีม IT ภายในองค์กร
Cloud

ใช้งานผ่าน Cloud

เริ่มใช้งานได้รวดเร็ว ลดภาระการดูแล Server เข้าถึงผ่าน Web Browser จากทุกที่

จุดเด่น

  • เริ่มใช้งานได้รวดเร็ว
  • ลดภาระการดูแล Server
  • ขยายระบบรองรับผู้ใช้งานหรือเอกสารจำนวนมากได้ง่าย

ข้อพิจารณา

  • มีค่าใช้จ่ายตามการใช้งาน
  • ต้องพิจารณานโยบายด้านการจัดเก็บข้อมูลบน Cloud
